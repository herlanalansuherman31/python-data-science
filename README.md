# 🐍 Python Data Analysis & Manipulation Portfolio

Repositori ini berisi kumpulan modul, proyek, dan studi kasus yang saya kerjakan untuk menguasai analisis data menggunakan **Python** dan ekosistem dasarnya (terutama **Pandas**). Di sini tercakup materi dari pemahaman dasar, manipulasi tabel kompleks, hingga visualisasi data.

l
## 🛠️ Tech Stack & Libraries
* **Bahasa Pemrograman:** Python 3.x
* **Library Utama:** * `pandas` (Data Manipulation & Cleaning)
  * `numpy` (Mathematical Operations)
  * Library pelengkap lainnya sesuai kebutuhan proyek (BeautifulSoup/Scrapy untuk Web Scraping, Matplotlib/Seaborn untuk Visualisasi).
