# 🐍 Python Data Analysis & Manipulation Portfolio

Repositori ini berisi kumpulan modul, proyek, dan studi kasus yang saya kerjakan untuk menguasai analisis data menggunakan **Python** dan ekosistem dasarnya (terutama **Pandas**). Di sini tercakup materi dari pemahaman dasar, manipulasi tabel kompleks, hingga visualisasi data.

## 📁 Struktur Repositori & Materi Pembelajaran

Repositori ini dibagi menjadi beberapa folder terstruktur sesuai dengan alur pembelajaran data analisis:

1. **01. Python Crash Course**
   * Fondasi dasar pemrograman Python sebelum masuk ke pengolahanu data.
2. **02. Intro to Pandas**
   * Pengenalan ekosistem Pandas: Series, DataFrame, membaca berbagai jenis file (CSV, Excel), dan inspeksi data awal.
3. **03. Project 1 - Web Scraping**
   * Proyek praktis mengambil (harvesting) data langsung dari halaman website untuk dijadikan dataset siap pakai.
4. **04. Filtering Data**
   * Teknik menyeleksi baris dan kolom berdasarkan kondisi tertentu (Boolean Indexing, `.loc`, `.iloc`).
5. **05. Data Extraction**
   * Teknik mengekstrak data spesifik dari teks, tanggal (datetime), atau sumber data mentah.
6. **06. Pivot Table**
   * Implementasi fungsi tabel silang (Pivot) di Pandas untuk meringkas informasi, mirip dengan fitur Pivot Table di Microsoft Excel.
7. **07. Project 2 - Data Visualization**
   * Proyek menyajikan pola data ke dalam bentuk grafik/chart (menggunakan library seperti Matplotlib atau Seaborn) agar lebih mudah dipahami oleh bisnis.
8. **08. GroupBy and Aggregation**
   * Mengelompokkan data berdasarkan kategori tertentu dan menghitung nilai statistik dasar (Mean, Median, Sum, Count).
9. **09. Merging and Concatenation**
   * Teknik menggabungkan beberapa tabel data yang terpisah menjadi satu kesatuan data menggunakan fungsi `merge()` (seperti SQL Join) dan `concat()`.

## 🛠️ Tech Stack & Libraries
* **Bahasa Pemrograman:** Python 3.x
* **Library Utama:** * `pandas` (Data Manipulation & Cleaning)
  * `numpy` (Mathematical Operations)
  * Library pelengkap lainnya sesuai kebutuhan proyek (BeautifulSoup/Scrapy untuk Web Scraping, Matplotlib/Seaborn untuk Visualisasi).
